# RegistryFlow CLI

A CLI tool for registry flow. Visit [registryflow.com](http://registryflow.com/) for more information.

## Installation

Install globally using npm:

```bash
npm install -g registryflow
```

Or using pnpm:

```bash
pnpm add -g registryflow
```

Or using yarn:

```bash
yarn global add registryflow
```

## Usage

After installation, you can use the `registryflow` command from anywhere:

```bash
registryflow init
```

## Commands

- `init` - Initialize a new project
